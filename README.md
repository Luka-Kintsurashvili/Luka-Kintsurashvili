    * **The Result:** Successfully visualized the "squeeze" effect where food costs began outpacing wage growth post-2020.

    * **The How:** Performed data cleaning and type conversion on revenue/budget columns. Used **NumPy** for conditional logic to categorize movies by "Era" and "Controversy Gap."
